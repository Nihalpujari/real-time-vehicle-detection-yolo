# 🚗 Real-Time Vehicle Detection using YOLO

## 📌 Overview

This project is a **real-time vehicle detection system** that uses a webcam or video input to identify and classify different types of vehicles such as cars, buses, trucks, and bikes.

The system processes live video frames and applies a deep learning model (YOLO) to detect vehicles and draw **bounding boxes with labels and confidence scores**.

---

## 🎯 Features

* 📸 Real-time detection using webcam
* 🚗 Detect multiple vehicle types
* 📦 Bounding boxes with labels
* ⚡ Fast and efficient detection (YOLO)
* 🎥 Works on both images and video

---
